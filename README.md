# LinkedList in Python

This repository contains Python implementations of all major Linked List data structures.
It is designed for learning, practice, and understanding the internal working of linked lists.

## Repository Structure

linked-list/
├── singly/
│   └── singly_linked_list.py
├── doubly/
│   └── doubly_linked_list.py
├── circular/
│   └── circular_linked_list.py

## Linked List Types Covered

### Singly Linked List
- Node creation
- Insertion operations
- Deletion operations
- Traversal

### Doubly Linked List
- Forward and backward traversal
- Insertion and deletion
- Previous and next pointer handling

### Circular Linked List
- Circular node connections
- Traversal without null values
- Insert and delete operations

## Language Used
- Python 3

## Purpose
- Learn linked list concepts
- Practice data structures in Python
- Prepare for interviews and exams

1. Clone the repository
2. Navigate to the project folder
